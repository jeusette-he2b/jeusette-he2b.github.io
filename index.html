Site réalisé lors du cours d'ISD

Bienvenue sur le site réalisé dans le cadre du cours d’Informatisation des services documentaires !

Sur ce site, vous pourrez consulter la documentation portant d'une part sur les réponses à deux questions essentielles, c'est-à-dire des questions dont les réponses impliquent les niveaux supérieurs de la taxonomie de Bloom. D'autret part, vous aurez accès au cahier des charges de mon TFE qui porte sur l’inventaire et la valorisation du Fonds Faecq. Dans cette partie, je reprends l’étude de l’existant, les contraintes et les expressions fonctionnelles, comme nous l’avons vu dans le cours d’informatisation des services.

De plus, le site reprend un dossier documentaire sur le « Resource Description Framework » (RDF).
